‚ö° HTML Best Practices

Essential Tips:

  • Always use DOCTYPE: <!DOCTYPE html> at the top
  • Set language: <html lang="en"> for accessibility
  • Use semantic elements: <header>, <main>, <footer> instead of generic <div>
  • Add alt text to images: Essential for screen readers
  • Use proper headings: H1 ‚Üí H2 ‚Üí H3 in hierarchical order
  • Validate your HTML: Use tools like W3C Markup Validator
  • Keep it clean: Indent properly and use meaningful names for IDs and classes

Common HTML Attributes:

  • id - Unique identifier for an element
  • class - CSS class name(s) for styling
  • href - URL for links
  • src - Source URL for images, scripts, etc.
  • alt - Alternative text for images
  • title - Tooltip text
  • target - Where to open links (_blank, _self)
  • placeholder - Hint text for form inputs

‚ö° CSS Cascade and Specificity

Understanding how CSS rules are applied when multiple selectors target the same element:

Element selector: Gray background, gray text

Class selector wins: Blue background, blue text

ID selector wins: Red background, red text

Specificity hierarchy (highest to lowest):

  1. Inline styles (style="...") - Specificity: 1000
  2. IDs (#myId) - Specificity: 100
  3. Classes (.myClass), attributes, pseudo-classes - Specificity: 10
  4. Elements (h1, p, div) - Specificity: 1

Important CSS concepts:

  • Cascade: When multiple rules apply, the last one wins (if same specificity)
  • Inheritance: Some properties are inherited from parent elements
  • !important: Overrides specificity (use sparingly!)

üìè CSS Units, Colors, and Positioning

Now let's explore specific CSS properties in detail!

%: Percentage relative to parent element

The child element is 50% width and 75% height of its parent container.

50% √ó 75% of parent

em: Relative to the font-size of the element

1em equals the font-size of the current element. Padding and borders scale with font-size.

Font-size: 18px, Padding: 1em (18px)
Nested: 2em font-size (36px), 0.5em padding (18px)

rem: Relative to the font-size of the root (<html>) element

1rem equals the root font-size (16px). Always consistent regardless of nesting.

Font-size: 1.5rem (24px), Padding: 1rem (16px), Margin: 2rem (32px)

vw, vh: Relative to viewport width/height

vw = 1% of viewport width, vh = 1% of viewport height. Resize your browser to see the effect!

50vw √ó 20vh (resize browser to see effect)
